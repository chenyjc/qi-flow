关于系统

量化交易策略系统是一个基于Qlib和FastAPI的专业量化交易策略回测平台，提供数据管理、模型训练、策略回测和结果分析等功能。

技术栈

后端

FastAPI + Qlib

前端

HTML5 + Tailwind CSS + Chart.js

数据

Qlib格式的A股数据

系统特点

  • 基于Qlib的专业量化交易策略回测
  • 支持多种模型训练算法（LightGBM、XGBoost、线性模型）
  • 提供详细的回测结果分析和可视化
  • 友好的用户界面和交互体验
